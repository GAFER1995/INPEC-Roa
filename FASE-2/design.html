Proposito del sitio

El propósito del sitio a crear es principalmente poder sistematizar la información que se adquiere mes a mes referente a la recepción de encomiendas las cuales ingresan al establecimiento. Para poder generar un sistema de información ordenado se pretende almacenar la información de cada encomienda, esto ayudara en gran medida a tener un sistema de base de datos completo e integral que pueda controlar el ingreso de elementos a cada persona sin que se evidencie ingresos excesivos, donde perdure la realización de una actividad trasparente y equitativa según el seguimiento de las normas establecidas para este establecimiento.
